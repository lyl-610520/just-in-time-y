# Just In Time - 恰逢其时

一个充满温度的PWA应用，帮助你记录生活中的每一个美好时刻。

## ✨ 功能特色

### 🌅 智能打卡系统
- **快捷打卡**: 起床打卡、睡觉打卡，一键记录生活节奏
- **自定义打卡**: 支持生活、学习、工作分类，记录每一个重要时刻
- **重复提醒**: 智能检测重复打卡，避免遗漏

### 🎨 极致美学体验
- **毛玻璃效果**: 现代化的UI设计，带来沉浸式视觉体验
- **液态玻璃**: 流畅的动画过渡，丝滑的交互感受
- **动态背景**: 根据时间和天气自动变化的背景系统
- **四季之树**: 随季节变化的装饰元素，增添生活气息

### 🌟 游戏化元素
- **宠物系统**: 可爱的宠物伙伴，陪伴你的每一天
- **花朵成长**: 通过打卡积累阳光值，见证花朵的成长历程
- **成就系统**: 丰富的成就解锁，激励持续使用
- **配饰收集**: 为宠物收集四季配饰，打造专属形象

### 🎵 音乐播放器
- **内置音乐**: 精选的治愈音乐，陪伴你的时光
- **自定义播放**: 支持上传个人音乐，打造专属播放列表
- **固定位置**: 导航栏上方的固定播放器，随时享受音乐

### 📊 数据统计
- **打卡分析**: 详细的打卡数据统计，了解生活规律
- **趋势图表**: 直观的图表展示，追踪个人成长
- **睡眠分析**: 专业的睡眠时长统计，关注健康作息

### ⚙️ 智能设置
- **主题切换**: 支持浅色/深色/自动主题，适配系统偏好
- **多语言**: 中英文双语支持，国际化体验
- **通知管理**: 智能提醒系统，不错过重要时刻
- **数据备份**: 本地数据存储，保护个人隐私

## 🚀 快速开始

### 环境要求
- Node.js 16.0+
- 现代浏览器（支持ES6+）

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 📱 PWA特性

- **离线运行**: 支持离线使用，数据本地存储
- **添加到主屏**: 一键添加到手机主屏幕
- **推送通知**: 智能提醒和通知系统
- **响应式设计**: 完美适配各种设备尺寸

## 🎯 使用指南

### 首次使用
1. 打开应用，等待加载完成
2. 允许位置权限，获取本地天气信息
3. 允许通知权限，接收智能提醒
4. 开始你的第一次打卡！

### 日常使用
- **早上**: 点击"我起床啦"，记录美好的一天开始
- **白天**: 使用自定义打卡，记录学习、工作、生活的点滴
- **晚上**: 点击"我要睡了"，记录一天的结束
- **查看**: 在统计页面查看你的打卡数据和成长轨迹

### 个性化设置
- 在设置页面调整主题和语言
- 自定义通知时间和提醒内容
- 设置倒计时事件，不错过重要时刻
- 管理音乐播放列表，打造专属音乐空间

## 🛠️ 技术架构

- **前端框架**: 原生JavaScript + ES6模块
- **构建工具**: Vite
- **PWA支持**: Vite PWA插件
- **动画库**: GSAP
- **图表库**: Chart.js
- **样式**: CSS3 + CSS变量 + 响应式设计
